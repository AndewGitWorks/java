# 🏦 ATM - Система банкомата

Полнофункциональное приложение банкомата на Java с графическим интерфейсом Swing. Поддерживает операции со счетом, платежи и администраторскую панель для просмотра истории транзакций.

## 🎯 Возможности

### Пользовательские операции
- 💰 **Снять наличные** - вывести деньги со счета
- 💵 **Пополнить счет** - пополнить баланс
- 📱 **Оплата телефона** - оплатить счет оператора
- 🏠 **Оплата ЖКХ** - оплатить коммунальные услуги
- 💳 **Перевод на карту** - перевести деньги на другую карту
- 👤 **Перевод пользователю** - перевести деньги другому пользователю
- ⚖️ **Оплата штрафов** - оплатить штрафы ГИБДД

### Администраторские функции
- 📊 Просмотр истории всех транзакций
- 🔍 Фильтрация по пользователям
- 📈 Аналитика платежей

## ⚙️ Требования

- **Java**: JDK 11 или выше
- **База данных**: SQLite (входит в проект)
- **ОС**: macOS, Linux, Windows

## 📦 Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd swing-atm-app
```

2. Убедитесь, что у вас установлена Java:
```bash
java -version
```

3. Откомпилируйте проект:
```bash
javac -d bin -sourcepath src src/**/*.java
```

## ▶️ Запуск

### Команда для запуска приложения

```bash
java --enable-native-access=ALL-UNNAMED -cp "bin:lib/sqlite-jdbc.jar" app.Main
```

### Или с альтернативной папкой компиляции (out):

```bash
javac -d out -sourcepath src src/**/*.java
java --enable-native-access=ALL-UNNAMED -cp "out:lib/sqlite-jdbc.jar" app.Main
```

### Параметры команды

- `--enable-native-access=ALL-UNNAMED` - разрешение на доступ к нативным методам SQLite
- `-cp` - classpath (пути к скомпилированным классам и библиотекам)
- `bin:lib/sqlite-jdbc.jar` - папка с классами и драйвер SQLite
- `app.Main` - главный класс приложения

### Работа с базой данных

#### Инициализировать БД

```bash
sqlite3 atm.db << 'EOF'
DROP TABLE IF EXISTS users;
DROP TABLE IF EXISTS transactions;

CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    full_name TEXT NOT NULL,
    card_number TEXT UNIQUE NOT NULL,
    pin TEXT NOT NULL,
    balance REAL DEFAULT 0
);

CREATE TABLE transactions (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    user_id INTEGER,
    type TEXT NOT NULL,
    amount REAL,
    description TEXT,
    created_at TEXT
);

INSERT INTO users (full_name, card_number, pin, balance) VALUES 
    ('Иван Петров', '1234567890123456', '1234', 10000.00),
    ('Мария Сидорова', '9876543210654321', '5678', 15000.00),
    ('Сергей Иванов', '5555666677778888', '9999', 5000.00);

.quit
EOF
```

#### Просмотреть таблицу пользователей

```bash
sqlite3 atm.db "SELECT * FROM users;"
```

#### Просмотреть все транзакции

```bash
sqlite3 atm.db "SELECT * FROM transactions ORDER BY created_at DESC;"
```

#### Просмотреть транзакции конкретного пользователя

```bash
sqlite3 atm.db "SELECT * FROM transactions WHERE user_id=1 ORDER BY created_at DESC;"
```

#### Проверить баланс пользователя

```bash
sqlite3 atm.db "SELECT full_name, card_number, balance FROM users WHERE id=1;"
```

#### Очистить базу данных

```bash
rm atm.db
```

## 🎨 Интерфейс

### Окно входа
- Ввод номера карты и PIN кода
- Кнопка "ВХОД" для пользователей
- Кнопка "АДМИН" для администраторов

### Личный кабинет пользователя
- Отображение баланса
- 8 кнопок для различных операций
- Удаленное обновление баланса в реальном времени

### Администраторская панель
- Таблица со всеми транзакциями
- Фильтр "Все операции"
- Фильтр "По пользователю"
- Выход из панели

## 🎯 Пример использования

### Вход как пользователь

1. Запустите приложение
2. На экране входа введите:
   - Номер карты: `1234567890123456`
   - PIN код: `1234`
3. Нажмите "ВХОД"
4. Выполняйте нужные операции
5. Нажмите "Выход" для выхода

### Вход как администратор

1. На экране входа нажмите "АДМИН"
2. Введите:
   - Имя: `admin`
   - Пароль: `admin`
3. Просмотрите историю транзакций
4. Используйте фильтры для поиска
5. Нажмите "Выход" для выхода

## 🔒 Безопасность

- Данные пользователей хранятся в локальной SQLite БД
- PIN коды хранятся в открытом виде (только для учебного проекта)
- Все операции логируются в БД
